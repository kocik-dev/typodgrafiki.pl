# CSS Houdini: Erstellen Sie benutzerdefinierte Effekte, die in regulärem CSS nicht verfügbar sind

Wollten Sie schon immer fortgeschrittene Effekte erstellen, die über die Möglichkeiten von regulärem CSS hinausgehen? CSS Houdini ist eine neue Technologie, mit der Sie benutzerdefinierte Effekte und Stile erstellen können, die zuvor unmöglich zu erreichen waren. Mit Houdini können Sie das Rendering-Verhalten von Browsern auf eine Weise steuern, die zuvor nur mit JavaScript möglich war.

## Was ist CSS Houdini?

CSS Houdini ist ein Set von API-Schnittstellen, mit denen Entwickler den CSS-Rendering-Mechanismus in Browsern "freischalten" und steuern können. Houdini eröffnet völlig neue Möglichkeiten für die Erstellung benutzerdefinierter Effekte, Animationen und Stile, die zuvor unerreichbar waren. Sie können zum Beispiel benutzerdefinierte CSS-Eigenschaften und Effekte erstellen und sogar die Elementdarstellung steuern, für die sonst JavaScript erforderlich wäre.
